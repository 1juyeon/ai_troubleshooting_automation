# SOLAPI SMS 발송 설정 가이드

## 📱 SOLAPI 개요

SOLAPI는 국내 최고의 SMS 발송 서비스로, 안정적이고 빠른 SMS 발송을 제공합니다.

## 🔑 API 키 발급 방법

### 1단계: SOLAPI 회원가입
1. [SOLAPI 공식 웹사이트](https://solapi.com/) 접속
2. 우측 상단 "회원가입" 클릭
3. 필요한 정보 입력 후 가입 완료

### 2단계: 프로젝트 생성
1. 로그인 후 대시보드 접속
2. "프로젝트 생성" 클릭
3. 프로젝트명 입력 (예: "PrivKeeper SMS")
4. 프로젝트 생성 완료

### 3단계: API 키 발급
1. 생성된 프로젝트 클릭
2. 좌측 메뉴에서 "API 관리" 선택
3. "API 키 생성" 클릭
4. API 키와 API Secret 확인 및 복사

### 4단계: 발신번호 등록
1. 좌측 메뉴에서 "발신번호 관리" 선택
2. "발신번호 등록" 클릭
3. 사용할 전화번호 입력 (예: 01012345678)
4. 인증 완료 후 발신번호 사용 가능

## ⚙️ 애플리케이션 설정

### 사이드바 설정
1. **SOLAPI API Key**: 발급받은 API 키 입력
2. **SOLAPI API Secret**: 발급받은 API Secret 입력
3. **발신자 번호**: 등록된 발신번호 입력

### 연결 테스트
1. "🔗 SOLAPI 연결 테스트" 버튼 클릭
2. 연결 성공 시 "✅ SOLAPI 연결 성공!" 메시지 확인

## 📱 SMS 발송 방법

### AI 분석 결과 페이지에서
1. AI 분석 완료 후 "📱 SMS 발송" 섹션 확인
2. 수신자 이름과 전화번호 입력
3. SMS 메시지 내용 확인/수정
4. "📱 SMS 발송" 버튼 클릭

### 이력 관리 페이지에서
1. 상세보기 버튼(🔍) 클릭
2. "📱 SMS 발송" 섹션에서 수신자 정보 입력
3. SMS 메시지 내용 확인/수정
4. "📱 SMS 발송" 버튼 클릭

## 💰 요금 안내

### 기본 요금
- **가입비**: 무료
- **월 기본료**: 무료
- **SMS 발송**: 건당 20원 (대량 발송 시 할인)

### 충전 방법
1. 대시보드에서 "충전" 메뉴 선택
2. 원하는 금액 선택 후 결제
3. 결제 완료 후 즉시 사용 가능

## 🔒 보안 주의사항

### API 키 보안
- API 키와 Secret을 안전하게 보관
- 타인과 공유하지 않음
- 정기적으로 갱신 권장

### 발신번호 관리
- 본인 명의의 전화번호만 사용
- 불법적인 용도로 사용 금지
- 수신자 동의 후 발송

## 🚨 문제 해결

### 연결 실패 시
1. API 키와 Secret 재확인
2. 인터넷 연결 상태 확인
3. SOLAPI 서비스 상태 확인

### SMS 발송 실패 시
1. 발신번호 등록 상태 확인
2. 잔액 확인
3. 수신자 번호 형식 확인 (01012345678)

### 지원 문의
- **고객센터**: 1544-5959
- **이메일**: support@solapi.com
- **운영시간**: 평일 09:00~18:00

## 🎯 최적화 팁

### 메시지 길이
- SMS는 90자 이내 권장
- 긴 메시지는 자동으로 분할 발송

### 발송 시간
- 업무시간(09:00~18:00) 발송 권장
- 긴급한 경우 즉시 발송 가능

### 수신자 관리
- 자주 연락하는 수신자 정보 저장
- 그룹별 발송 기능 활용
